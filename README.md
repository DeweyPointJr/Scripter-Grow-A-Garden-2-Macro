# Scripter Grow A Garden 2 Macro
## Installation:
1. Download [AutoHotKey v1.1](https://autohotkey.com)
2. Install the macro from the [latest release](https://github.com/DeweyPointJr/Scripter-Grow-A-Garden-2-Macro/releases/latest)
3. Extract the zip file

## Usage:
1. Join Grow A Garden 2 on Roblox
2. Use the menu in the macro to select which items to buy, and press start
3. Press F3 (or another keybind if you changed it) to stop the macro.
